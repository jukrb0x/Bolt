---
title: "bolt version"
---

Print the current Bolt version.

## Usage

```bash
bolt version
```

## Description

Prints the installed version of Bolt. Use this to verify your installation or when reporting issues.

## Options

None.

## Examples

```bash
bolt version
# Output: 1.2.2
```
